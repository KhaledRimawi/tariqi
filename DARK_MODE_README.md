# ميزة الوضع المظلم - Dark Mode Feature

## نظرة عامة
تم إضافة ميزة الوضع المظلم إلى تطبيق Tariqi لتحسين تجربة المستخدم وتوفير راحة أكبر للعيون في الإضاءة المنخفضة.

## المميزات المضافة

### 1. سياق الوضع المظلم (Dark Mode Context)
- ملف: `src/contexts/DarkModeContext.js`
- يدير حالة الوضع المظلم على مستوى التطبيق
- يحفظ تفضيلات المستخدم في localStorage
- يطبق التغييرات تلقائياً على عنصر body

### 2. زر التنقل (Toggle Button)
- ملف: `src/components/DarkModeToggle.js`
- زر أنيق مع أيقونات القمر والشمس
- يظهر في شريط التنقل العلوي
- يدعم الشاشات المحمولة

### 3. التنسيقات المحدثة
- ملف: `src/Navbar.css` - محدث لدعم الوضع المظلم
- ملف: `src/index.css` - أنماط عامة للوضع المظلم
- ملف: `src/pages/Home.css` - أنماط الصفحة الرئيسية
- ملف: `src/components/DarkModeToggle.css` - أنماط الزر

## كيفية الاستخدام

### للمستخدمين
1. ابحث عن زر الوضع المظلم في شريط التنقل العلوي
2. اضغط على الزر للتنقل بين الوضع العادي والمظلم
3. سيتم حفظ تفضيلك تلقائياً

### للمطورين
```javascript
// استخدام السياق في مكون
import { useDarkMode } from './contexts/DarkModeContext';

function MyComponent() {
  const { isDarkMode, toggleDarkMode } = useDarkMode();
  
  return (
    <div className={isDarkMode ? 'dark-mode' : ''}>
      {/* محتوى المكون */}
    </div>
  );
}
```
